Для реализации сервера был использован фреймворк netty, в котором уже есть встроенные компоненты для работы с http-запросами, поэтому основная работа заключалась в реализации собственного обработчика, который получает уже упакованные объекты HttpRequest. В обработчике достаётся url-строка и с помощью QueryStringDecoder достаётся путь и параметры. Для отправки ответа были созданы небольшие классы HtmlResponseCreator и Table для создания html-таблиц.
